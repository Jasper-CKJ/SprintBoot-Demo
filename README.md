# SprintBoot-Demo

## 项目简介
`SprintBoot-Demo` 是一个用于学习并实战 `Spring Boot` 的项目

## 开发环境
- **JDK 1.8 +**
- **Maven 3.5 +**
- **Mysql 5.7 +**
